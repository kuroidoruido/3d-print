# BuJo Helpers

Tool to made faster monthly pages of my Bullet Journal.

Includes:
- main calendex page frame
- date helper
- half circle tool from 1cm to 8cm diameter

## Print Setting

- Printer brand: Creality
- Printer: Ender 3
- Rafts: Yes
- Supports: Yes
- Resolution: 0.15mm
- Infill: 20
- Filament brand: Amazon Basics
- Filament color: Red
- Filament material: PLA
