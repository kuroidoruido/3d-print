# Shower hook

Simple shower hook to hang towel.

## Print Setting

- Printer brand: Creality
- Printer: Ender 3
- Rafts: No
- Supports: No
- Resolution: 0.3mm
- Infill: 20
- Filament brand: Amazon Basics
- Filament color: Grey
- Filament material: PLA

## Notes/Tips

For a shower from the brand Ideal Standard.
